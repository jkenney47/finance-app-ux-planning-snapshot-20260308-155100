# Blocking Questions

Ask only when missing information blocks MVP scoping, sequencing, or acceptance criteria. Ask at most 3 questions.

## Priority Questions

1. Which user role is the single primary user for MVP?
2. What is the non-negotiable deadline or launch window for MVP?
3. Which constraint is strictest: timeline, budget, compliance, or reliability?
4. What must not break in the current system during MVP rollout?
5. Are there mandatory integrations for MVP, or can they be stubbed first?
6. Are there data retention, privacy, or audit constraints that affect design?

## Selection Rule

- Prefer questions that change MVP boundaries or milestone order.
- Avoid asking for implementation details the agent can decide safely.
