# Triage Questions

Ask only what is needed to avoid bad assumptions.

## Preferred 3 Questions

1. What outcome matters most for you: speed, reliability, or cost?
2. Is there anything we must not break while doing this?
3. Is there a target date or deadline?

## Optional Follow-Ups

- Should this be rolled out gradually or all at once?
- Are there legal, compliance, or data handling constraints?
- Should this optimize for mobile, web, or both?
