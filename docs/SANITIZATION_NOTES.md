# Sanitization Notes

This snapshot was curated for public-safe UX planning.

Excluded on purpose:
- secrets and environment files
- backend services and database code
- CI/CD, workflow, and deployment configuration
- internal operations surfaces and automation instructions
- agent tooling and local memory files
- private Git history and branch structure

Included on purpose:
- screen-level source snapshots
- shared visual components
- theme context
- product and UX intent summaries
- baseline screenshot evidence

This snapshot should be treated as a UX planning artifact, not as an operational or deployable repository.
