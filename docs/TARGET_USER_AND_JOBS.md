# Target User And Jobs

## Primary User

A non-technical consumer user who wants straightforward financial direction without jargon, dense financial tooling, or intimidating advisor language.

## Main Jobs To Be Done

- Help me understand my financial situation quickly.
- Help me see what I should do next.
- Help me trust the recommendation instead of feeling like it is arbitrary.
- Help me understand what I gain by following a recommendation.
- Help me move forward without needing to connect everything immediately.

## Emotional Needs

- reassurance
- clarity
- control
- momentum
- reduced anxiety

## Common Risks In The UX

- feeling like the app is asking for too much too early
- feeling judged or overwhelmed
- seeing too much financial detail before understanding what matters
- seeing polished visuals without a clear action path
- being pushed into account linking before trusting the product

## UX Standard

Every major screen should make the user feel:
- I understand what this screen is for
- I know what matters here
- I know what to do next
